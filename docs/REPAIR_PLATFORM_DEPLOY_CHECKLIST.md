# Repair Platform Deployment Checklist

1. Configure the existing Mahar Shwe Repair Tracking web-app URL.
2. Run Prisma migration deployment.
3. Run server syntax checks and frontend build.
4. Test local Repair ID generation.
5. Import a known Mahar Shwe Repair ID.
6. Link a provider Repair ID to a partner-shop local job and sync it.
7. Link an IMEI or serial and verify device history.
8. Confirm tenant integrity reports zero violations.
