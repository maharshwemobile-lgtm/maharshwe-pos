# Repair ID Format

The Repair workspace exposes one Repair ID only.

Accepted existing code pattern:

```text
MS0551
AC0001
HH0001
MH0001
PO0001
BO0001
TL0001
P0001
```

No shop code, year, month, referral ID or second public Repair ID is shown. Provider linkage and PostgreSQL UUIDs stay internal.
